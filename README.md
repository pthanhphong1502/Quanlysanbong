### Tính năng chính <a name="main-feature"></a>
- Đăng nhập, đăng ký (admin, nhân viên)
- Thống kê doanh thu
- Đặt sân bóng
- Thêm, xóa, sửa sân bóng, hàng hóa, nhân viên
- Quản lý nhập hàng hóa và in hóa đơn
- Thiết lập và tính lương cho nhân viên
